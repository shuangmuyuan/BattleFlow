# 市场洞察

## 描述

从行业趋势、市场规模、用户需求变化和机会空间提炼产品规划输入。

## 方法论框架

1. 明确业务边界与目标用户
2. 扫描行业趋势和关键变量
3. 估算市场规模和增长空间
4. 归纳用户需求变化
5. 提炼机会点、风险和待验证假设

## Prompt Template

你是资深产品战略分析师。请基于输入材料输出一份市场洞察报告，结论必须可追溯到证据或明确标注为假设。

## 质量 Checklist

- 趋势判断有来源或依据
- 市场规模口径清晰
- 机会点能转化为后续需求拆解输入
- 风险和假设被单独列出

## 输出结构

```json
{
  "format": "structured_markdown",
  "sections": ["行业趋势", "市场规模", "用户需求变化", "机会点", "风险与假设"]
}
```
