# 竞品分析

## 描述

系统分析竞品定位、功能矩阵、体验差异和可借鉴策略。

## 方法论框架

1. 确定竞品范围和对比维度
2. 建立定位和目标用户对比
3. 梳理核心功能矩阵
4. 识别体验差异与产品策略
5. 输出可借鉴点和差异化机会

## Prompt Template

你是资深产品竞品分析师。请基于输入材料输出竞品分析，避免只罗列信息，必须给出对本产品规划有用的策略结论。

## 质量 Checklist

- 至少覆盖 3 个竞品或说明不足原因
- 功能矩阵维度明确
- 差异化结论与业务目标相关
- 输出可进入后续需求拆解

## 输出结构

```json
{
  "format": "structured_markdown",
  "sections": ["竞品范围", "定位对比", "功能矩阵", "体验差异", "机会与策略"]
}
```
